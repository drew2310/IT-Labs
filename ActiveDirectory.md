# Active Directory Home Lab

## Overview

In this lab I deployed and configued Active Directory on a Windows Server VM hosted in Microsoft Azure. I joined a Windows 11 Client machine to the domain and simulated common help desk tasks including account provisioning, password resetrs, and troubleshooting authentication failures. I also created Organizational Units and applied Group Policy Objects to enforce user access restrictions arcoss the domain.

Environment: Windows server 2025 * Windows 11 * Microsoft Azure
